Monte Carlo

  • RunPostProcessor - Static method to run a single Monte Carlo Post Processor
  • RunPostProcessors - Static method to run multiple Monte Carlo Post Processors
  • RunSimulation - Static method to run a single Monte Carlo simulation
  • RunSimulations - Static method to run multiple Monte Carlo simulations

Solvers

  • AbsorbedEnergyOfRhoAndZ - Absorbed Energy as a function of rho and z
  • FluenceOfRhoAndZ - Fluence as a function of rho and z
  • PHDOfRhoAndZ - Photon Hitting Density in cylindrical coordinates
  • PHDOfRhoAndZTwoLayer - Photon Hitting Density in cylindrical coordinates for a two-layer tissue
  • ROfFx - Spatial-frequency reflectance
  • ROfFxAndFt - Reflectance as a function of spatial- and temporal- frequencies
  • ROfFxAndT - Reflectance as a function of spatial-frequency and time
  • ROfFxTwoLayer - Reflectance as a function of spatial-frequency for two-layer tissue
  • ROfRho - Reflectance as a function of rho
  • ROfRhoAndFt - Reflectance as a function of rho and temporal-frequency
  • ROfRhoAndFtTwoLayer - Reflectance as a function of rho and temporal-frequency for two-layer tissue
  • ROfRhoAndT - Reflectance as a function of rho and time
  • ROfRhoAndTimeTwoLayer - Reflectance as a function of rho and time for a two-layer tissue
  • ROfRhoTwoLayer - Reflectance as a function of rho for two-layer tissue
  • SetSolverType - Sets the solver type

Spectroscopy

  • GetOP - Returns the optical properties
